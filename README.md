# Build/Run Instructions

## Build Instructions

1. npm i
2. npm run builddb
3. Add the following enviroment var to .env in the application folder

|  variable   |   value    |
| :---------: | :--------: |
|   DB_HOST   | localhost  |
|   DB_USER   |    root    |
| DB_PASSWORD | dbpassword |
|   DB_NAME   |   dbname   |

## Run Instructions

1. npm start
